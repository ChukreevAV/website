Projects

Descriptions of various projects which were carried out by of with help of GIS-LAB authors.

Conservation

Mapping boundaries of "Volga Delta" Ramsar wetland (rus)
Картографический проект по нанесения границ ценной природной территории по официальным данным для дальнейших принятий решений. Все существующие опубликованные схемы ВБУ изобилуют ошибками.

Environmentally important objects of Northern Caspian Sea region (rus) (under development )
Разрабатываемый проект, являющийся развитием предыдущего, охватывает весь Северный Каспий и показывает текущую ситуацию с расположением ценных природных территорий (ООПТ, ВБУ) и потенциальных, реальных угроз для них.

Ornithology

Identification of nesting biotopes of Bolshezemeskaya tundras' Jack Snipe using high resolution remote sensing data (rus)
Результаты проекта по картированию гнездопригодных биотопов гаршнепа с помощью данных Landsat-7/ETM+.

Finding intersection of two lines using known points and angles (biangulation)
Решение с помощью ГИС типичной задачи постобработки данных радиотелеметрии (не только птиц, любых движущихся объектов).

Minimum convex polygon construction with consideration of locational error
При построении участка обитания по данным телеметрии, ошибка локации часто не учитывается. Эта статья объясняет как учесть этот показатель при построении участка обитания.

Smoothing movement tracks with consideration of locational error
Ошибку локации также можно учитывать при построении трека перемещения объекта. Траектория перемещения без учета ошибки локации обычно включает много ошибочных локаций, расстояние между которыми меньше, чем ошибка локации.

«Important Bird Areas (IBA) of Western Siberia which are important for bird conservation» map series (rus)
Описательная информация о серии карт, посвященных ключевым орнитологическим территориям международного значения (Important Bird Areas, IBAs) в Западной Сибири. На этой странице дается исчерпывающая информация об использованных данных, методологии, проекции и источниках получения картографической информации.

Internet mapping for presenting satellite telemetry data from Lesser White-fronted Geese (Anser erythropus)
Data on sattelite tracking of White-fronted Geese is available for viewing on the basis of remote sensing data using Google Maps API.
