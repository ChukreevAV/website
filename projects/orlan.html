Интернет-картографирование для визуализации данных спутниковой телеметрии белоплечего орлана (Haliaeetus pelagicus )

Описание проекта использующего технологии Google Maps для визуализации результатов.

При обработке первичных данных телеметрии на всех этапах использовались инструменты Argos-tools.


Интерактивные схемы перемещения

Результаты спутникового слежения за белоплечими орланамидоступны на базе картографического сервиса Google Maps. Карты обновляются по мере получения данных о перемещениях птиц. В качестве картографической основы используются данные Landsat и QuickBird, доступные через картографический сервис Google Maps. По картам можно осуществлять навигацию (увеличение, уменьшение, перемещение) (подробнее о возможностях Google Maps), а также просмотреть дату, время и класс точности каждой локации

Выберите одну из ссылок приведенных ниже, чтобы перейти к интерактивной схеме перемещения соответствующей птицы

Внимание - загрузка данных локаций может занять некоторое время, в течение которого карта не отображается. Для схем оставлены только локации классов 0-3 (подробности о классах локации Argos - здесь)
