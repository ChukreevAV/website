Нарезка растра на номенклатурные листы с помощью ERDAS Imagine

Использование Mosaic Tool и AOI для создания номенклатурной нарезки данных в ERDAS Imagine.

Нарезка растра на отдельные листы задача непростая и возникающая сравнительно часто. Возможности Erdas позволяют заметно упростить ее решение. Ниже будет описан способ “нарезки” растров с помощью инструмента Mosaic Image. Описывается способ нарезки: с помощью AOI (областей интереса), хотя возможно применение и других инструментов, например, Map Series Tool (инструмент создания серий карт).

Идея такова – имеется шейп-фаил номенклатурной разграфки карт 100 000 масштаба (масштаб может быть любой), необходимо из этого фаила создать несколько AOI (“областей интереса”), и по созданным AOI нарезать растр на листы 100 000 масштаба.

Данная процедура невыполнима функцией Subset Image (можно использовать одну AOI за один запуск Subset Image), но вполне осуществима средствами Mosaic Image.

Описание состоит из нескольких частей:

  1. Создание AOI на основе шейп-фаила
  2. Нарезка растра с помощью Mosaic Image и пользовательской AOI.
