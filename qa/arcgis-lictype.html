Определение типа используемого продукта ArcGIS

Как определить, какой тип лицензии установленного пакета ArcGIS

Пакет ArcGIS, в зависимости от типа лицензии, может быть представлен в виде одного из 3 типов продуктов (в порядке возрастания функциональности):

  • Arcview - базовые возможности картографирования, создания отчетов и картографического анализа
  • Arceditor - функциональность больше чем у ArcView, поддерживает возможность построения баз геоданных.
  • Arcinfo - максимальная функциональность, наиболее мощное приложение ArcGIS Desktop, включающее Arcinfo Workstation и множество дополнительных инструментов.

Для того, чтобы определить тип используемого продукта и соответственно функциональность, необходимо запустить ArcMap и обратить внимание на заголовок окна, в нем, после названия программы (ArcMap, ArcCatalog) должен также быть обозначен тип продукта, например:

Второй способ получить информацию о функциональности установленного пакета - использовать ArcGIS Desktop Administrator.

Для того, чтобы его запустить, необходимо из меню Start (Пуск) выбрать:

Start\Programs\ArcGIS\Desktop Administrator

Далее, на закладках можно посмотреть тип используемого продукта (Software product) и подробную информацию о лицензиях (Availability).
