Онлайн-сервис по конвертации систем координат данных

Краткая аннотация

Данный сервис предназначен для пересчета данных находящихся в одной системе координат, в другую. На данный момент присутствуют только широко распространенные системы координат, с которыми чаще всего приходится сталкиваться. Пересчет осуществляется с помощью библиотеки Proj, одной из самых развитых и свободно распространяемых систем для пересчета координат.

Для того, чтобы перевести ваши значения из одной системы координат в другую, выберите исходную и конечную систему координат, введите ваши данные и нажмите на кнопку Пересчитать.

Если в списках отсутствует нужная вам система координат, напишите и мы ее добавим.
