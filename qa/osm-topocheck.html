Сервис проверки наличия ошибок

Эта страница предназначена для быстрой проверки на наличие топологических ошибок в данных OpenStreetMap XML. Проверяется только полигональная геометрия. Все ошибки подобного рода с возможностью быстрого перехода к исправлениям можно посмотреть на общей карте.

Сервис является вспомогательным для общей ежедневной проверки и призван решить проблему быстрой перепроверки типа "решено/не решено". Его можно использовать для быстрых экспериментов с геометрией и отношениями на предмет того, как на них реагирует валидатор.

Проверяемые данные OSM

Обычно представляют собой объект или набор объектов сохраненных в формат OSM с помощью, например, JOSM.

Что нужно сделать с файлом OSM

1. Использовать форму внизу, чтобы загрузить файл (не более 3 мегабайт)

2. Изучить полученный отчет на предмет наличия ошибок. Если сообщений об ошибках нет, значит в полигональных данных их обнаружено не было.
