Installing .NET extensions for ArcGIS

How to install .NET libraries

1. Download and install the .NET Framework version 1.1 redistributable package which includes everything you need to run applications developed using the .NET Framework (23 Mb)
http://www.microsoft.com/downloads/...

2. Insert ArcGIS Desktop installation disk into the CD-drive.

3. If ArcGIS windows doesn't appear after it, locate startup.exe file on your disk and run it.

4. Chose "Install ArcGIS Desktop" from the list.

5. Choose "Modify" on the next page.

6. Select ".NET Support" to be installed on local hard drive. Click "Next" to install .NET support. You won't see ".NET Support" in the list, if you haven't downloaded and installed .NET Framework (see 1 )
