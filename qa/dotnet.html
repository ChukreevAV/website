Установка поддержки .NET для ArcGIS

Необходимые библиотеки для работы в ArcGIS некоторых приложений

Ошибка "Unable to get installer types in the c:\program files\gis-lab\...\*.dll assembly.---> One or more of the types in the assembly unable to load" возникающая при установке некоторого программного обеспечения, в том числе расширений для ArcGIS означает, что у вас не установлена поддержка .NET.

Для ее установки и включения проделайте следующие действия:

1. Загрузите и установите пакет платформы .NET версии 1.1 (23 Mb)
http://www.microsoft.com/downloads/...

2. Вставьте диск с дистрибутивом ArcGIS Desktop в CD-привод.

3. Если окно ArcGIS не появилось, найдите и запустите файл startup.exe с этого диска.

4. Выберите из списка "Install ArcGIS Desktop".

5. На следующей странице выберите "Modify".

6. Включите опцию ".NET Support". Нажмите Next для установки поддержки .NET. Внимание! Если платформа NET (п.1) на вашем компьютере не установлена, этот пункт отсутствует, перейдите к пункту 1.

7. Для корректной работы расширений рекомендуем установить (переустановить после установки поддержки NET) Service Pack 3 или более новую версию.
