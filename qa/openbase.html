Открытые базы геоданных PostGIS

Перечь открытых для чтения баз данных, содержащихся в них данных и настроек доступа.

Данная страница содержит перечень полезных баз геоданных к которым можно обратиться через сервер GIS-Lab. Базы используют PostgreSQL/PostGIS. Данные из баз можно загружать любым ПО умеющим это делать (QGIS, gvSIG, MapServer).

Параметры доступа:

  • Сервер: gis-lab.info
  • Имя: guest
  • Пароль: guest
  • Имя базы: [GEN|Geosample|OSM]

Если вы используете QGIS, вы можете скачать настройки всех баз, загружаемые в менеджер баз данных.

Помимо баз, все данные также доступны и в формате Shape, подробное описание данных и данные для скачивания на локальный компьютер можно найти если перейти по ссылке "подробнее".

Пример настройки доступа к базе OSM в QGIS

Доступны следующие базы:

GEN

Эта база содержит различные слои, которые могут пригодиться в повседневной работе, в частности:

  • Номенклатурные сетки-разграфки карт масштабов 1:1М, 1:500К, 1:200К, 1:100К (подробнее)
  • Границы субъектов РФ сделанный на базе GIS-Lab (подробнее)
  • Границы административного деления РФ на уровне страны, федеральных округов, субъектов, муниципальных районов на базе OSM, с обрезкой по морю и без (подробнее)
  • Центры муниципальных районов и городских округов на базе OSM (подробнее)
  • Слои с населенными пунктами VMap0 с переведенной на русский язык атрибутикой (подробнее)
  • Разграфка 5х5 градусов для SRTM в формате GeoTIFF и других целей (подробнее)
  • Разграфка 1х1 градусов для SRTM в формате HGT и других целей (подробнее)
  • Разграфка WRS-1 и WRS-2 для данных Landsat (подробнее)
  • Схема зон UTM/GK (подробнее)
  • Разграфка продуктов MODIS (подробнее)

Geosample

Эта база содержит результаты проекта Geosample, удобного для освоения ГИС и обучения себя и других. Проект Geosample содержит данные и сформированные проекты по четырем субъектам РФ. Доступны следующие слои:

  • Административные границы
  • Населенные пункты
  • Гидросеть линейная
  • Гидросеть полигональная
  • Объекты (POI)
  • Дорожная сеть
  • Железные дороги
  • Охраняемые природные территории
  • Растительность
  • Почвы
  • Экорегионы

Подробнее о Geosample.

OSM

В этой базе содержатся ежедневно обновляемые выгрузки данных OpenStreetMap по РФ и странам бСССР, а также отдельно по субъектам РФ.

Подробнее о данных OSM.
