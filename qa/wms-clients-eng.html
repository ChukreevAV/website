Using WMS data in desktop GIS

How to make use of WMS data

Let say you have a link to some WMS service, this article describes what you need to do to make use of this link and load data using this service in your GIS software. We show how tos for 3 widely used software, starting from free one. Most of the other commercial and free GIS software have ability to work with WMS.

  1. Using WMS in:
    • QGIS
    • ArcGIS
    • ArcGIS Explorer

QGIS

WMS-layer in QGIS is loaded using menu option Layer\Add WMS layer. Click on New button in the window and enter parameters of the connection. Most often the needed parameters are arbitrary name of the service and its URL, which often includes signs "&" or "?" in the end of the line.

You can try this examples:

http://gis-lab.info/cgi-bin/wmsworld?
http://gis-lab.info/cgi-bin/wmsopenifl?

Click OK and then Connect. After the connection is set, choose layers and click Add. Layers will be added to the list of layers of the main map window.

If there are several possible options for graphic formats, try PNG first, as the background will be transparent, this helps.

ArcGIS

In ArcGIS you can load WMS-layers through Add Data window. After opening this window, select GIS Servers under Look in: in the top right area of the window and hit WMS Servers, which will ask you for the URL of the WMS.

After you've entered URL click Get Layers and select needed layers.

ArcGIS Explorer

WMS-layers in ArcGIS Explorer are opened as any other data using File\Open... menu,where you need to switch to Servers section and hit WMS. In the new dialog, enter URL of WMS.

After clicking Ok, you will be asked to select some parameters, such as raster resolution, layers used and zoom level after layers are loaded.

As WMS serves data in raster format, on smaller scales you can see pixelization effect - this is normal.
