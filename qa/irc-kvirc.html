Настройка KVIrc для канала #gis-lab

Краткое описание и настройка для KVIrc

IRC-канал GIS-Lab предназначен в большей степени для совместного обсуждения проектов и в меньшей для решения мелких персональных вопросов и задач, для которых рекомендуется использовать Форум. Перед первым использованием рекомендуется сделать тестовый вход, чтобы не потерять время на настройку клиента в процессе самого обсуждения.

KVIrc – это свободный IRC-клиент, доступный для GNU/Linux, Microsoft Windows и Mac OS X. Чтобы поставить канал #gis-lab в автозагрузку, нужно выполнить следующие операции.

1. В меню Настройки выбираем Настроить KVirc.


2. Слева выбираем пункт Список серверов и находим сеть Freenode.


3. Раскрыв список серверов сети, выбираем любой из них. В данном примере – chat.eu.freenode.net. После чего жмем кнопку Дополнительно.


4. В следующем окне выбираем вкладку Идентификация и прописываем ник и если ник зарегистрирован – пароль.


5. Переходим на вкладку Соединение и отмечаем пункт Подключаться к этому серверу при старте и выбираем Использовать кодировку сети.



6. Переходим на вкладку Входить на каналы. Прописываем канал #gis-lab и жмем кнопку Добавить.


7. Сохраняем настройки и перезапускаем KVIrc.
