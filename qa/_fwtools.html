FWTools - руководство пользователя

Сборник советов по работе с набором FWTools

Данная статья - сборник советов для тех, кто хочет воспользоваться какими-то из инструментов FWTools, так как этот пакет управляется исключительно из программной строки, некоторые моменты требуют дополнительного объяснения для тех, у кого опыт работы с ней невелик. В пакет FWTools входит несколько пакетов:

  • GDAL/ORG
  • Proj
  • OpenEV
  • Mapserver
  • OGDI
  • Python
