Объединение линий

Инструмент для объединения двух пересекающихся линий в одну

1. Замечания по установке

Расширение доступно из репозитория ГИС-Лаб.

JoinLines разработан для QGIS версии 1.0.0 и старше, для работы не требует наличия дополнительных модулей.

Исходный код модуля можно получить выполнив команду

svn co http://svn.gis-lab.info/joinlines joinlines

2. Работа c расширением

После подключения расширения на панели появится кнопка , запускающая процесс объединения.

Для использования:

  1. загрузить линейный слой в QGIS и сделать его активным (выделить)
  2. выделить две линии, которые нужно объединить. ВАЖНО! Линиии должны быть созданы с небольшим перехлестом в конце или при включенном прилипании (т.е. иметь общую точку)
  3. нажатием на кнопку выполнить объединение

В отличие от инструмента «Объединить выбранные объекты», JoinLines выполняет только попарное объединение и выдает не комплексную геометрию (т.н. MultiLine), а простую.
