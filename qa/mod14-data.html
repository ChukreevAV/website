Данные по тепловым аномалиям MOD14: получение

Получение данных

Общая информация

Продукты MODIS группы Тепловые аномалии/Пожары (Thermal Anomalies/Fire) - MOD14 дают информацию об очагах пожаров, т.е. о локациях горящих в данный конкретный момент (Рис. 1). Принцип детектирования пожаров использует тот факт, что они сильно излучают в среднем инфракрасном диапазоне.

Общую информацию о MOD14 и других продуктах этой серии можно найти в статье "Данные по тепловым аномалиям MOD14A1: описание и получение".

Эта статья предоставляет доступ к зеркалу данных по территории Российской Федерации и сопредельным территориям.

Основная часть данных получена в результате коллективного проекта в котором участвовали: bolotoved: 2003, 2006, 2007, 2009, 2011, amuriy: 2004, aesapronov: 2005, _DR_: 2010, Максим Дубинин: 2000, 2001, 2008, Voltron: 2002.

Рис. 1. Пример сгоревшей территории (черная область) и очаги MOD14A1 (красные точки)

Получение данных

Данные доступны бесплатно. Формат данных - HDF. Получить оригинальные данные можно через систему WIST (Warehouse Inventory Search Tool). Подробное описание работы с системой можно прочитать здесь. Следует иметь в виду, что данные поступают несколько раз в день и поэтому здесь может сказываться ограничение WIST в 2 тысячи гранул на один заказ. Таким образом, на всю территорию России, например, можно заказать данные только за один месяц.

Для заказа можно использовать следующие предварительно сформированные файлы *.query разбитые на полумесячные промежутки с 2000 по 2011 г. Область заказа - вся РФ за исключением Чукотки. Скачать файлы запросов и скрипт для генерации дополнительных файлов.

Данные без прохождение процедуры регистрации, заказа и ожидания также можно на GIS-Lab. Покрытие доступных данных доступных через наш сервер приведено на Рисунке 2 (также можно скачать полигон запроса в формате shape).

Рис. 2. Зеркало данных MOD14/MYD14 на сервере GIS-Lab

База данных

Данные также доступны через БД PostGIS. БД можно подключать в QGIS и любом другом ПО понимающем такие базы данных.

Настройки:

  • Сервер: gis-lab.info
  • Имя: fire_usr
  • Пароль: J59DY
  • Имя базы: mod14
