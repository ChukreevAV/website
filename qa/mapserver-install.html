Установка MapServer для Windows

В статье даются краткие сведения о установке MapServer для Windows.

Описание MapServer

О том, что такое MapServer, и каковы его возможности, сказано в статье Создание картографических сервисов с использованием MapServer. Введение. Дистрибутивы MapServer (для различных платформ) можно скачать по адресу: http://mapserver.gis.umn.edu/download/current/. В этой статье рассматривается установка MapServer для Windows (MS4W).

Процедура установки MapServer

  1. Скачиваем дистрибутив MapServer, представляющий собой zip-архив.
  2. Находим внутри архива файл README_INSTALL. Знакомимся с содержимым. Наиболее важным представляется следующее: во-первых, пакет устанавливает Apache, PHP, MapServer CGI, MapScript (как следствие, предварительно устанавливать Internet Information Services - IIS, - нет необходимости); во-вторых, устанавливать пакет нужно в корень диска (не обязательно системного), если же установить его не в корневую папку, то могут возникнуть проблемы.
  3. Как было сказано, нужно распаковать архив в корень диска, например, диска "D". В результате получаем каталог с MapServer: D:/ms4w.
  4. Устанавливаем MS4W Apache Web Server (для этого предоставляется специальный bat-файл: D:/ms4w/apache-install.bat). Запускаем на выполнение этот файл и он начнет установку Apache Web Server как службу Windows. В процессе установки bat-файл должен выдать следующие сообщения:

    Installing the Apache MS4W Web Server service
The Apache MS4W Web Server service is successfully installed.

Testing httpd.conf....
Errors reported here must be corrected before the
service can be started.
The Apache MS4W Web Server service is starting.
The Apache MS4W Web Server service was started successfully.


  5. Все. Теперь Apache MS4W Web Server service установлен и будет автоматически запускаться после перезагрузки компьютера. Убедиться в том, что этот сервис запущен и работает можно через Панель управления\Администрирование\Службы. Среди сервисов должен появиться этот сервис, тип запуска - авто.

    Согласно инструкции, при работе с Windows 95,98,ME запустить Apache как сервис не удастся, поэтому каждый раз нужно будет вручную вызывать D:/ms4w/Apache/bin/httpd.exe.
  6. Проверяем, как работает Apache: открываем браузер, вводим адрес http://localhost/ (или, что даст тот же результат, http://127.0.0.1/). Если все в порядке, должна появиться главная страничка MS4W.
  7. Установка завершена. Так же можно установить дополнительные приложения, например, MapLab, ka-Map и/или поддержку Oracle 10g, SDE 9.1, скриптов CSharp, Java, Python, организовать парольный доступ к создаваемым приложениям и т.д. Как это делается, описано все в том же README_INSTALL.
