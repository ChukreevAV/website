1. Описание данных

Отделом "Технологии спутникового мониторинга" Института Космических Исследований Российской Академии Наук (ИКИ РАН) предоставлены данные по пройденным огнем площадям на территорию России полученные на основе спутниковых данных Terra/Aqua-MODIS за 2010 год. В данных отсутствуют пройденные огнем площади на территории сельскохозяйственных угодий.

Предоставленные данные предлагается использовать например для сравнения с результатами сторонних и разрабатываемых алгоритмов детектирования сгоревших площадей. Предоставленные данные имеют актуальность лето 2011 года и могут изменяться (перерасчитываться) в будущем с изменением методологии.

Эти же данные только для просмотра незарегистрированным пользователям и с большими правами для пользователей прошедших официальный процесс регистрации доступны на сайте ФГУ "Авиалесоохрана"в рамках Информационной системы дистанционного мониторинга (ИСДМ).

Данные представляют собой растровое изображение, в котором закодирован день обнаружения пожара: 0 - пожара в области, соответствующей пикселю не было, 1 - пожар возник в первый день года (1 января), 2 - пожар возник во второй день года и т.д. Система координат - широта/долгота, WGS84.

Охват: север: 83:00:03.6N; юг: 39:59:56.4N; запад: 17:59:56.4E; восток: 179:59:56.4W. Пространственное разрешение пикселя исходной растровой матрицы: 7.2x7.2 секунд.

3. Получение данных

Для удобства использования растр был векторизован. Данные можно скачать в формате ESRI Shape. Размер архива 16 Мб.

Поле value содержит дату регистрации пожара.
