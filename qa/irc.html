Настройка IRC для канала #gis-lab

Краткое описание и настройка для различных клиентов

IRC-канал GIS-Lab предназначен для координации усилий по проектам и свободного обсуждения персональных вопросов и задач. Если вам никто не ответил в IRC рекомендуем также обратиться в Форум. Перед первым использованием рекомендуется сделать тестовый вход, чтобы не потерять время на настройку клиента в процессе самого обсуждения.

Адрес канала: irc://irc.freenode.net/#gis-lab

1. Общая информация

Сервер: irc.freenode.net
Канал: #gis-lab
Используемая кодировка: UTF-8 (как настроить)

Зайти прямо сейчас без установки клиента.

2. Использование через Веб-браузер: CGI

Самый простой способ. Для того чтобы присоединиться к обсуждению, нужно просто перейти на специальную страницу и ввести любые имя и пароль.

3. Использование через Miranda

Удобный способ позволяющий не устанавливать нового ПО, если у вас уже есть Miranda. Для начала работы (Miranda\IRC\Join a channel) необходимо произвести следующие настройки:

Miranda\Options\IRC\Advanced\
Server code page: UTF-8

Настройка автовхода:

Miranda\Options\IRC\Advanced\
Enable: включить
Perform on the event: FreeNode
Ввести в область под этим: /joinm #gis-lab (нажать +)

4. Использование через ChatZilla

Удобный способ для браузера Mozilla Firefox, позволяющий использовать специальный plugin. Чтобы начать слушать IRC, нужно его скачать и установить, включить (Tools\ChatZilla) и ввести в строку адреса:  irc:// irc.freenode.net/#gis-lab.

К сожалению, QIP общение в кодировке UTF-8 не поддерживает, поэтому участие с помощью этого клиента в IRC невозможно.
