Простой способ изменения системы координат данных в ArcGIS

Один из способов как изменить систему координат данных в ArcGIS

Пожалуй, самый простой способ изменить систему координат векторных или растровых данных с созданием нового набора данных с нужной системой координат.

1. Открываем ArcCatalog. В ArcCatalog выбираем файл набора данных (вектор или растр), который требуется перепроецировать (двойной щелчок или правой кнопкой и Properties). В окне Properties находим раздел Spatial Reference: если система координат не определена (undefined) нажимаем Edit и задаём требуемую систему.

Наример для данных SRTM, обработанных и представленных в виде грида:
Select, из списка выбрать Geographic Coordinate Systems\World\WGS 1984.prj

2. Открываем ArcMap и загружаем наш набор данных (в данном случае растр SRTM) и в свойствах слоя проверяем, что Spatial Reference именно та, которую мы только что задали.

Выбираем правой кнопкой свойства Data Frame (строка над всеми слоями, объединяющая их в группу, с жёлтым значком и по умолчанию с надписью “Layers”) - Properties. Выбираем вкладку Coordinate System и в разделе “Select a coordinate System” выбираем ту систему координат, в которую мы хотели бы перевести наш набор данных. Для SRTM, к примеру, выбираем:

Predefined -> Projected Coordinate Systems -> Utm -> Wgs 1984 -> Zone X (нужная зона). Например, зона 36 – Московская область.

Данная операция означает, что мы задали фрейму данных проекцию UTM, все данные будут автоматически трансформированы в новую систему координат, однако эта СК для данных сохраняется только в проекте ArcMap. Для создания нового набора данных в новой системе координат нужно выполнить пересчет слоя. Если после задания новой системы координат изображение пропало, возможно, новая система координат выбрана неверно.

3. Экспортируем набор данных в новой системе координат. Для этого правой кнопкой нажав на нужный слой в списке слоёв выбираем  Data -> Export Data. В открывшемся окне ставим переключатель в Spatial Reference на Data Frame (Current). Проверяем Name и Location, тип выходных данных и экспортируем растр.

Результатом является новый набор данных в необходимой нам системе координат. На приведенном ниже примере показан растр до и после изменения системы координат (системы координат соответствуют приведенным выше).
