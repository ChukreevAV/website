Введение

Этот инструмент может применяться для того, чтобы использовать один файл стиля (qml) для нескольких растровых или векторных слоев в QGIS. Расширение находится в разработки и протестировано в QGIS 1.0.1.

Получение

Расширение может быть установлено с помощью Plugin Installer (Plugins\Fetch python plugins...), где должен быть добавлен и выбран 3rd party репозиторий:

Name: GIS-Lab
URL: http://gis-lab.info/programs/qgis/qgis-repo.xml

Plugin Installer будет сообщать об обновлениях расширения.

Исходный код может быть получен из SVN

svn co http://svn.gis-lab.info/multiqml multiqml

 

Демонстрация

В этой демонстрации расширение используется, чтобы применить единую легенду для серии одинаково индексированных растров, имеющих одинаковую цветовую таблицу. Также показывается как можно применять легенду типа min-max для серии ЦМР USGS.
