Добавление пользовательских систем координат в ERDAS IMAGINE

Краткая аннотация

Данное описание показывает процесс добавления новых систем координат, как спроектированных (прямоугольных), так и географических. Многие из этих операций можно также выполнить через графический интерфейс ERDAS IMAGINE.

Это описание также иллюстрирует, как можно определить параметры трансформации, которые использует ERDAS, это может быть полезно тем, кто интересуется, какие же все-таки цифры стоят за названиями "Pulkovo 1942", "Pulkovo 1942-1 (Russia)" и т.д.

Добавление новой группы систем координат

 

Добавление нового датума

Для того, чтобы добавить новый набор параметров трансформации, необходимо:

1. Найти файл описаний эллипсоидов (сфероидов), обычно хранящийся в папке установки ERDAS IMAGINE\etc\spheroid.tab

2. В файле описаний найти соответствующий эллипсоид. Например, найдем эллипсоид Красовского.

"Krasovsky" {
	16 6378245.0   6356863.0188
	"Krasovsky" 0 0 0 0 0 0 0
	"Afgooye" -43 -163 45 0 0 0 0
	"Hanoi 1972" -17.51 -108.32 -62.39 0 0 0 0
	"DE_42/83 to ETRS89" 24.9 -126.4 -93.2 3.054e-07 1.197e-06 1.988e-07 1.01e-06
	...
}

3. Добавить строку новым набором параметров. Допустим, мы хотим добавить набор параметров трансформации соответствующий ГОСТ 51794-2001 (подробнее о настройке GPS c этим набором, подробнее о переводе данных в эту СК). В данном примере показано использование 3 параметров для данной трансформации, возможно, вам понадобится 7. 

"Krasovsky" {
	16 6378245.0   6356863.0188
	"Krasovsky" 0 0 0 0 0 0 0
	"Afgooye" -43 -163 45 0 0 0 0
	"Hanoi 1972" -17.51 -108.32 -62.39 0 0 0 0
	"Pulkovo-GOST" 23.92 -141.27 -80.9 0 0 0 0
	...
}

4. Сохранить файл.

Добавление новой системы координат в существующую группу

Для того, чтобы добавить новую систему координат в существующую группу, необходимо:

1. Найти файл описания группы систем координат. Он должен находится в папке установки ERDAS IMAGINE\etc\projections. Даный каталог содержит все группы систем координат, доступные в данный момент через графический интерфейс.

2. Изменить данный файл, добавив в него новую систему координат. Допустим, нам необходимо добавить географическую систему координат в группу Geographic. Изучим содержание файла geogrpahic.plb. Для добавления нужно указать:

  • Название новой СК
  • Эллипсоид
  • Набор параметров трансформации (датум)

Например, добавим новую систему координат на базе эллипсоида Красовского, но новым набором параметров трансформации (датумом), который мы создали на предыдущем этапе:

"Lat/Lon (Pulkovo GOST)" {
 INTERNAL 0 "Krasovsky" "Pulkovo-GOST" 0

}

В результате файл *.plb примет следующий вид. 

NAME "Geographic"
"Lat/Lon (Clarke 1866)" {
 INTERNAL 0 "Clarke 1866" "Clarke 1866" 0

}
"Lat/Lon (WGS 84)" {
 INTERNAL 0 "WGS 84" "WGS 84" 0

}
"Lat/Lon (Pulkovo GOST)" {
 INTERNAL 0 "Krasovsky" "Pulkovo-GOST" 0

}

3. Сохраненный файл geographic.plb можно оставить в папке ERDAS IMAGINE\etc\projections, а можно переместить в папку, где хранятся пользовательские наборы проекций USER\.imagineXXX\projections\. Второй вариант предпочтительнее, так как в этом случае будет легко переместить созданный набор(ы) для обновлении ERDAS или переходе на другой компьютер.
