Виртуальные глобусы

Краткая аннотация

  • GeoPortal - разработка Европейского Космического Агенства (ESA) и FAO, виртуальный глобус создан для обеспечения доступа к пространственным данным ESA.
  • Free Earth - кросс-платформенный глобус, не требующий клиента кроме браузера и внедряемый в другие веб-страницы с помощью Javascript API
  • OSSIM - Open Source Software Image Map. Открытый исходный код. Данные дистанционного зондирования.
  • Leica TITAN (12.2006)
  • CitySurf Globe (12.2006)
  • Microsoft Virtual Earth (November 2006)
  • Dapple Earth Explorer (July 2006)
  • Wayfinder Earth (beta, 2006) - виртуальный глобус для мобильных телефонов от Teleatlas, картографическая информаци.
  • 3D Weather Globe & Atlas by Software MacKiev (2005)
  • ESRI ArcGIS Explorer (2005)
  • Volvo Ocean Race Virtual Spectator (2005)
  • Erdas Imagine Virtual GIS (2005)
  • Google Earth (2005)
  • Global-i (~2005)
  • Punt (2005)
  • Ping 3map (2005)
  • EarthSLOT (2004)
  • osgPlanet (2004)
  • ESRI ArcGlobe (2003)
  • NASA World Wind (2003)
  • Eingana (2000 to ~2003)
  • Keyhole Earthviewer (June 2001/now Google Earth)
  • SINTEF Virtual Globe (2001/now Norkart Virtual Globe since 2006)
  • Celestia (2001)
  • Skyline Software TerraSuite (~2001)
  • GeoFusion GeoPlayer (company formed 2001)
  • SRI Terravision (2000)
  • EarthBrowser (Lunar Software Earthbrowser) - виртуальный глобус, который появился в 1998, в настоящий момент представляет из себя интерфейс к 15-тиметровым данным Landsat, регистрация платная ($23.95).
  • Microsoft MapPoint Virtual Globe (1998 now Windows Live Local)
  • Planet 9's VirtualEarth (1995)
  • Mark Pesce's WebEarth (1995)
  • Deutche Telekom's T-Vision (1995) (shown at Siggraph in August, not web based)
  • Blue Marble's Talking Globe CD (2D)(1994)
  • Hipparchus (~1992)
