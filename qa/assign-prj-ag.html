Назначение системы координат слою в ArcGIS

Прописывание системы координат в ArcCatalog с созданием файла prj.

Файлы описаний систем координат хранятся в специальных файлах с расширением prj. Эти файлы используются для обозначения систем координат для геоданных.

Эта статья описывает по шагам как "прописать" систему координат слою с помощью ArcCatalog. Результатом прописывания будет являться тот же слой, но к нему добавится файл prj с именем, как у файлов слоя.

Если вашей системы координат нет в списке готовых, нужно сначала ее создать. То, как это делается описывается в статье "Добавление пользовательских проекций в ArcGIS".

Для назначения системы координат слою, необходимо:

1. Открыть ArcCatalog, найти слой, щелкнуть по нему правой кнопкой мыши и перейти к его свойствам (Properties).

2. В окне свойств нужно выбрать закладку XY Coordinate System:

3. Собственно выбор назначаемой системы координат может осуществляться тремя способами.

Можно выбрать систему координат из уже существующих нажав Select. Разумеется, нужно заранее знать в какой системе координат находятся ваши данные, чтобы не назначить неправильную. В появившемся окне нужно выбрать нужную систему координат.

В комплект ArcGIS входит большое количество предустановленных систем координат, географических и спроектированных (прямоугольных). Все они хранятся в виде отдельных файлов prj в специальной папке Coordinate Systems.

Систему координат слоя можно также импортировать (подхватить) с другого слоя, для котого она уже прописана (Import).

Если нужной системы координат после нажатия Select не обнаружилось, можно создать ее самому выбрав кнопку New.
