2. Получение

Поиск

Поиск и отбор данных CORONA осуществляется используя интерфейс EarthExplorer. Чтобы найти данные нужно:

  1. Выбрать тип данных
  2. Ввести дополнительные критерии поиска и географическую область с помощью карты или посредством ввода координат.
  3. Нажать кнопку поиск.

Для поиска, можно выбрать две коллекции данных в группе Declassified data:

  • Declass 1 (1996)
  • Declass 2 (2002)

Год в названии коллекции соответствует дате, когда данные стали доступными а не году съемки.

Примечание: по умолчанию поиском возвращается 10 сцен, для увеличения количества результатов, нужно выбрать большее значение на первой странице настройки поиска.

После некоторого времени, будет получен результат. Отметьте, что количество найденных сцен сразу не показывается, для того, чтобы его получить, нужна нажать на знак вопроса и оно будет расчитано.

Для перехода к просмотру данных необходимо выделить нужные наборы и нажать кнопку Results.

Отбор

  • N - порядковый номер в таблице результатов;
  • Download - N/A если данные нельзя скачать, если можно - ссылка;
  • Entity ID - внутренний идентификационный номер сцены;
  • Acquisition Date - дата съемки;
  • Mission Nbr - номер миссии;
  • Revolution - виток орбиты;
  • Frame Nbr - номер кадра;
  • Camera Resolution - камера и разрешение;
  • Camera Type - тип камеры;
  • Image Type - тип изображения (BW - черно-белое);
  • NW Corner - северо-западный угол (формат: DD°MM'SS.SS"N, DD°MM'SS.SS"E , например: 45°03'28.80"N, 43°50'20.40"E);
  • NE Corner - северо-восточный угол;
  • SW Corner - юго-западный угол;
  • SE Corner - юго-восточный угол;
  • Film Type - тип пленки;
  • Polarity - поляризация;
  • Generation - поколение;
  • Data Source - источник данных;
  • Footprint Annotation - подпись;
  • Prod Media Type - тип носителя.

Заказ

Стоимость одной сцены: 30 USD, независимо от ее типа.

Иногда попадаются сцены, которые можно скачать сразу (видимо это сцены, которые кто-то уже купил).
