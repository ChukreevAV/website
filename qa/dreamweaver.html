Настройка Dreamweaver для работы с GIS-Lab

Для тех, кто получил доступ для редактирования одного из разделов.

Самым удобным способом совместного редактирования, практически исключающим одновременное исправление одних и тех же файлов является редактирование страниц непосредственно на сервере. Для того, чтобы работать с GIS-Lab таким образом, необходимо следующим образом настроить свое соединение.

1. Открыть окно управления сайтами Site\Manage Sites...

2. Создать новый сайт New...FTP&RDS Server

3. Настроить соединение

В появившемся окне добавится новая запись имеющая вид ftp:/имя_сайта

После выбора сайта, Dreamweaver подключится к нему и в окне Files выведет список доступных документов, папок. Выбор осуществляется путем двойного щелчка на документе в этом списке. После сохранения редактирования документ автоматически обновляется на сервере.

Если дерево с файлами не открыто, необходимо либо выбрать закладку Files, либо выбрать из меню Window\Files либо F8.

Локальная копия не сохраняется (точнее сохраняется только в кэш, но использовать ее из кэша не рекомендуется).
