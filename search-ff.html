Поиск по сайту GIS-Lab.info для Firefox

При загрузке этой страницы должно появиться окно предлагающее добавить поисковый движок по сайту GIS-Lab.info в панели быстрого поиска Firefox. Необходимо нажать Add/Добавить, также можно включить немедленное использование, чтобы Firefox сделал этот движок активным.

После добавления, поиск по сайту можно осуществлять не заходя на него, просто вводом в эту область окна.

Если автоматического добавления поискового движка не произошло, нажмите на стрелку рядом с текущим поисковым движком и выберите из меню Add "GIS-Lab search".

Чтобы убрать добавленный движок, нажмите на ту же стрелку, нажмите Manage Seach Engines, выберите GIS-Lab.info и нажмите Remove.
